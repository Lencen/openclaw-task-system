工具规范与规则
1. 工具安装规则
  • 内置工具：OpenClaw 内置工具无需安装，直接可用
  • 插件工具：通过安装插件扩展工具能力
  • MCP 工具：通过 MCP 协议连接外部服务提供的工具
  • 权限配置：在 openclaw.json 中配置工具权限
查看工具文档
2. 工具安全使用规范
  • 权限最小化：使用 allow/deny 列表限制工具权限
  • 输入验证：所有输入参数必须验证
  • 审计日志：所有工具调用都会被记录
  • 敏感操作：exec 等敏感工具需要谨慎使用
3. 工具自动化调用规则
  • 频率限制：遵守 API 调用频率限制
  • 重试机制：失败时指数退避重试
  • 超时设置：设置合理的调用超时时间
  • 结果缓存：相同请求缓存结果
4. 工具开发规范
  • SDK 使用：使用 OpenClaw Plugin SDK 开发
  • 接口规范：定义清晰的输入输出类型
  • 文档完整：提供完整的工具描述和使用示例
  • 错误处理：完善的错误处理和提示
