🎯 系统简介

Task System 是一个基于 AI Agent 协作的任务管理系统，旨在通过自动化工作流和智能调度， 提升团队的协作效率和项目管理能力。系统采用"人机协作"的设计理念，让 AI Agent 成为团队的智能助手。

建设目标

  • 🎯 自动化工作流 - 任务自动识别、分配、执行、验收 查看详情 →
  • 🤖 智能Agent协作 - 多Agent协同工作，专业分工
  • 📊 全局可视化 - 任务、资源、进度实时监控
  • 🔄 流程规范化 - 标准化工作流程，沉淀最佳实践

详细流程说明

阶段 描述 负责人
需求输入 用户通过对话、邮件、文档等方式提出需求 用户
任务识别 AI自动分析需求，识别并创建任务 Main Agent
任务拆解 将复杂任务拆解为可执行的步骤 Deep Agent
Agent分配 根据任务类型智能分配合适的Agent Auto Executor
任务执行 Agent按照步骤执行任务，实时更新进度 Coder/Chat/Fast
自动测试 执行自动化测试，验证任务质量 Test System
自动验收 检查任务完整性、文档齐全性 Auto Validator

🚀 使用方式

1. 任务管理

通过自然语言对话与系统交互，系统自动识别并创建任务。

示例："请帮我完成XX功能开发，需要包含..."

2. 项目追踪

在项目页面查看项目进度、关联任务、团队成员工作量。

3. 知识查询

通过文档中心或直接对话查询知识库中的文档和信息。

4. 系统监控

在监控页面查看系统健康状态、任务执行情况、告警信息。

🤖 Agent角色分工

Agent 角色 主要职责
🧠 Main 领导者 任务识别、分配决策、整体协调
💻 Coder 执行者 代码编写、调试、技术实现
🔍 Deep 分析师 深度分析、方案设计、调研评估
💬 Chat 沟通者 对话交互、文档撰写、内容生成
⚡ Fast 快速执行 快速响应、简单任务、批量操作

🏗️ 技术架构

  • 前端: HTML5 + CSS3 + Vanilla JS
  • 后端: Node.js + Express
  • AI引擎: OpenClaw Gateway + 多模型支持
  • 数据存储: JSON文件系统（可扩展数据库）
  • 消息推送: WebSocket实时通信
